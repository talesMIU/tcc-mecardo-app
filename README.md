TCC - Microfrontend no mercado financeiro

Como rodar o app:

1- Baixar para um repo local. exemplo git clone 

2- Instalar cada um dos mfes, dentro da pasta root:

  2.1-  cd auth
        npm install
        npm run start
  2.2-  cd financa
        npm install
        npm run start
  2.3-  cd mercadoapp
        npm install
        npm run start
  2.4-  cd container
        npm install
        npm run start
        
A aplicação irá rodar em localhost:8080
